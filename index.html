How JavaScript Works

JavaScript is a high level programming language. High level language means this language is very softly coded. You do not need to know much about your machine for this. It has data type, operator, some important object (which you can use all the time) and function or method. JavaScript is currently the most popular scripting language. Although JavaScript was first introduced as a client site scripting language, it is now widely used in both client sites and server sites. Even Node js JavaScript runtime makes it possible to run JavaScript without a browser. But now it runs on any operating system except browsers and it has all the features of other high level programming languages.

Difference Of Local Storage Vs Session Storage

When a new document is loaded on a tab in the browser, some data is set as key and value somewhere in the Devtool of that browser. It is again set in two ways, one is local storage and the other is session storage. Session storage data can be removed by closing the browser or tab. Local storage or session storage can be set a new with the setItem key and value can also be accessed again with getItem.
